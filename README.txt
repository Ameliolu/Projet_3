2) Installation


Ce jeu nécessite l'ensemble des fichiers contenus dans le dossier "Projet 3".

Pour fonctionner, ce jeu nécessite également la bibliothèque Pygame. Pour être certain de disposer des bons éléments, ouvrez un 
terminal et exécutez la commande suivante :

pip install -r requirements.txt

----------------------------------------------------------------------------------------------------------------------------------

3) Fonctionnement


Vous pouvez lancer ce jeu en double-cliquant sur le fichier "mac_gyver_labyrinth.py" ou en exécutant celui-ci par un terminal.

Le but du jeu est simple. Mac Gyver doit récupérer les 3 objets se trouvant dans le labytrinthe pour pouvoir triompher du garde.

Le déplacement du personnage se fait via les touches directionnelles du clavier.

Le ramassage des objets se fait automatiquement en déplaçant le personnage dessus.
